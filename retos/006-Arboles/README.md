# Llegando a los árboles

Partiendo de lo que conocemos (Listas creadas a partir de nodos) que permite crear una estructura de datos lineal, y atendiendo que existen otras estructuras no lineales que habitualmente representan jerarquías:

- La familia.
- Una organización.
- El [DOM](https://es.wikipedia.org/wiki/Document_Object_Model).

Desarrolle la modificación que considera debe hacerse a lo que conocemos para poder representar estas estructuras no lineales.
